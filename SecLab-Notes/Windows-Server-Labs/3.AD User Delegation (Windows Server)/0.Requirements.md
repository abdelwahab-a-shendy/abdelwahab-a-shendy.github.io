### ✅ The IT staff member should be able to: 

1. Create new user accounts inside the designated OU.
    
2. Reset passwords for existing users.
    

### ❌ The IT staff member must NOT be able to:

1. Delete any user accounts.
    
2. Modify or create security groups.
    
3. Change OU permissions.
    
4. Perform any administrative actions outside the delegated tasks.
