### Steps to create a unified background across the domain :

1. Prepare the background image
		1. Prepare the image (for example: wallpaper.jpg).
		2. Place it in a shared folder on the Domain Controller or any server.
	2. Make it available to users as read-only:
		1. Right click → Properties → Sharing → Advanced Sharing → Share.
		2. Permissions → Everyone → Read ✅.
	
	![](Photos/Pasted%20image%2020250908002133.png)
		
		
- 2- Create a Group Policy
	- Open Group Policy Management (GPMC).
	- Right-click the Domain (NTI.local) → Create a GPO in this domain, and Link it here.
	- Name it, for example: Domain_Background :
	
3. Edit the policy
	1. Right-click the GPO (Domain_Wallpaper) → Edit.
	2. From the Group Policy Management Editor, go to:
	3. User Configuration
		1. → Administrative Templates
		2. → Desktop → Desktop
	4. Right-click on Desktop Wallpaper.
		
	5. Choose: Enabled.
		1. In Wallpaper Name, enter the full path to the image:

![](Photos/Pasted%20image%2020250908004155.png)
		
1. In Wallpaper Style → choose (Fill / Stretch / Center). => Apply → OK.
		
2. Don't forget to link the GPO to the OU
