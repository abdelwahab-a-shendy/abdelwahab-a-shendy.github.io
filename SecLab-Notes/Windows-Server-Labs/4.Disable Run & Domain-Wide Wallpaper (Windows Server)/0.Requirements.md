### ✅ The lab should achieve the following:
 
1. **Disable Run Command**
    
    - Users in the domain (or specific OU) should **not be able to open the Run dialog** (Win + R).
        
2. **Unified Desktop Background**
    
    - All users in the domain should have a **standardized wallpaper**.
        
    - The wallpaper image must be **read-only** for users.
        
    - The wallpaper should be **applied via Group Policy**.
        

### ✅ Implementation constraints:

- All settings must be applied **through Group Policy Objects (GPOs)**.
    
- Changes must be **applied to users automatically**, without manual configuration on each client.
    
- Security permissions on the wallpaper file must prevent users from modifying it.
