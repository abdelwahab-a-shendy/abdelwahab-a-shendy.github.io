
Before starting the hands-on lab, you’ll need to access your **Kibana instance** for the lab environment.\
This is done using the login credentials stored in a special file on your lab machine.

***

### ⚙️ **Steps:**

1. Open **Strigo Lab** and select the machine:

   ```bash
   lab-machine
   ```

   (This is the system containing your working files.)

2. Navigate to the following path:

   ```bash
   /home/student
   ```

3. Display the file contents:

   ```bash
   cat CREDENTIALS
   ```

4. You’ll see your login details 👇

   * **Username:** usually fixed (commonly `elastic`)

   * **Password:** changes for each lab

***

### 🔑 **Logging into Kibana**

1. Copy your **lab access URL** (usually starts with `https://i-xxxxx.elasticlabs.training/`)

2. Open it in a **new browser tab**

3. From the main screen, select **Kibana Instance**

4. The login page will appear:

   * Enter the **Username**

   * Paste the **Password** you found in the `CREDENTIALS` file

5. Click **Login** ✅

🎉 Now you’re inside the **Kibana environment** and ready to start your first hands-on lab in the **Elastic Stack**!
