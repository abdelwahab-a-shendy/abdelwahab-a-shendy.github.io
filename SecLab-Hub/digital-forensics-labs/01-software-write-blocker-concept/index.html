    🔹 What is a Write Blocker?

    It’s a tool—either hardware-based or software-based—that prevents any attempt to add, modify, or delete data on a disk or device during forensic analysis.\ In this guide, we focus on the software version.

    🔧 Tools Used:

    • Software Write Blocker from: forensicsoft.com/products/safeblock

    • The program is available with a 7-day trial.

    • Some Linux forensic distributions, like SIFT, come with a built-in software write blocker.

    ⚙️ Steps to Use the Write Blocker Program:

    1. Download and Install:

    2. Register on the website and download the software.

    3. You’ll be asked for a license file during installation.

    4. Set a password to protect the program from unauthorized use.

    5. Launch the Program:

    6. Upon launch, you’ll be prompted for the password.

    7. The interface will show:

      • Physical disks

      • Logical partitions (like C:\ or G:)

    8. Select Target Disk:

    9. You cannot apply blocking to the system partition (like C:).

    10. Choose another drive (e.g., G:) for testing.

    🧪 Practical Test:

    Before Activation:

    • You can copy or delete files freely from the G:\ drive.

    • You can read/write as you like.

    After Activation (Enable Blocking):

    • The program may close File Explorer if it's using the disk.

    • A 🔒 icon appears next to the drive name → indicating "write protection" is active.

    Test:

    • Try copying a file: ❌ Not allowed

    • Try deleting a file: ❌ Not allowed

    ✅ Result: The disk is now completely write-protected.

    ✅ Why Use a Write Blocker?

    Benefit Explanation
    🧩 Evidence Protection No modifications are allowed during analysis.
    🦠 Malware Defense Even if malware is present, it can't alter data.
    🔒 Data Integrity Evidence remains exactly as it was when seized.
    📸 Forensic Imaging Images created from the disk are legally trustworthy.

    📌 Important Note:

    Software Write Blockers are not recommended for critical cases since they are not as secure as hardware write blockers.\ However, they are useful for training or low-risk scenarios.

    🧭 Summary:

    Item Details
    Tool Used Software Write Blocker from ForensicSoft
    Purpose Prevent disk modification during examination
    State After Activation No copying, deleting, or editing files allowed
    Ideal Use Case Creating a forensic image or analyzing a suspect’s drive

    🛡️ شرح استخدام Software Write Blocker في التحليل الجنائي الرقمي

    ❗ الهدف من الـ Write Blocker هو حماية الأدلة الرقمية من أي تعديل أثناء الفحص، سواء عن طريق الخطأ أو البرامج الخبيثة.

    🔹 ما هو الـ Write Blocker؟

    هو أداة (سواء عتادية أو برمجية) تمنع أي محاولة للإضافة أو التعديل أو الحذف على القرص أو الجهاز أثناء تحليله.\ في هذا الشرح نركز على النسخة البرمجية (Software).

    🔧 الأدوات المستخدمة:

    • برنامج Software Write Blocker من موقع:\ forensicsoft.com/products/safeblock

    • يمكن تحميل البرنامج بنسخة تجريبية (7 أيام).

    • بعض توزيعات لينكس فورنزك مثل SIFT تحتوي على Software Write Blocker مدمج.

    ⚙️ خطوات استخدام برنامج Write Blocker:

    1. تحميل البرنامج وتثبيته:

    2. التسجيل في الموقع وتحميل البرنامج.

    3. سيُطلب منك ملف الترخيص (License File) أثناء التثبيت.

    4. أدخل كلمة مرور عند التثبيت لحماية البرنامج من الاستخدام غير المصرح به.

    5. تشغيل البرنامج:

    6. عند تشغيل البرنامج، يُطلب منك كلمة المرور.

    7. سيعرض لك:

      • الأقراص الفيزيائية (Physical Disks)

      • الأقسام المنطقية (Partitions) مثل C:\ أو G:\

    8. اختيار القرص المستهدف:

    9. لا يمكن عمل Block للقسم الذي عليه نظام التشغيل (مثل C:).

    10. اختر قرصًا آخر (مثال: G:) لإجراء التجربة.

    🧪 تجربة عملية:

    قبل التفعيل:

    • يمكنك نسخ أو حذف الملفات على القسم G:\ بسهولة.

    • تقدر تكتب و تشيل من عليه اللي انت عايزو

    بعد التفعيل (Enable Blocking):

    • البرنامج سيغلق مستكشف الملفات (Explorer) إذا كان يستخدم القرص.

    • تظهر علامة 🔒 أمام اسم القرص → تدل على أن "الكتابة ممنوعة".

    تجربة:

    • حاول نسخ ملف: ❌ لا يمكنك.

    • حاول حذف ملف: ❌ لا يمكنك.

    ✅ النتيجة: التأكيد أن القرص مغلق للكتابة بالكامل.

    ✅ لماذا نستخدم Write Blocker؟

    الفائدة الشرح
    🧩 حماية الأدلة لا يُسمح بأي تعديل على القرص أثناء الفحص.
    🦠 منع تأثير البرمجيات الخبيثة حتى لو كانت هناك فيروسات، لا يمكنها التعديل.
    🔒 الحفاظ على سلامة البيانات الأدلة تبقى كما كانت وقت الضبط تمامًا.
    📸 ضمان نسخ Forensic نظيفة عند إنشاء Image من القرص، تكون موثوقة قانونيًا.

    📌 ملحوظة مهمة:

    لا يُفضَّل استخدام Software Write Blockers في الحالات الحرجة؛ لأن الحماية البرمجية ليست بقوة الحماية العتادية (Hardware Write Blockers).\ ومع ذلك، يُستخدم للتدريب أو في الحالات الأقل خطورة.

    🧭 الخلاصة:

    العنصر التوضيح
    الأداة المستخدمة Software Write Blocker من ForensicSoft
    الهدف منع التعديل على القرص أثناء الفحص
    الوضع بعد التفعيل لا يمكن نسخ أو حذف أو تعديل أي ملف على القرص
    الحالة المثالية عند نسخ Image أو تحليل قرص مشتبه به
